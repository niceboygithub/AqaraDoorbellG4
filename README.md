# homeassistant
## G4
After enabled telnet, you can use (AlexxIT's webrtc)[https://github.com/AlexxIT/WebRTC] to add Aqara DoorBell G4 to HomeAssistant

## G410
The camera part support RTSP, you can easily enable RTSP to add to homeassistant without any modification.
But for the gateway part, you need to enable to telnet to replace mosquitto to public.
